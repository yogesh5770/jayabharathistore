### 2. Add a Number
1.  Click the **Blue "+" Button** (or "Add a new Caller ID").
2.  **Select Country**: Change the flag to India (+91).
3.  **Enter Phone Number**: Type the number you want to allow (e.g., `7010196231`).
4.  **Verify Method**: Choose **SMS** (Text Message).
5.  Click **"Verify Number"**.

### 3. Enter the Code
1.  Twilio will send a 6-digit code to that phone.
2.  Enter that code on the screen.
3.  Click **Submit**.
